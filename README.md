# Urban Brew – Specialty Coffee & Work Café Website

Urban Brew is a modern, responsive café website concept designed for freelancers, students, and remote professionals.

## Features
- Clean and minimal UI design
- Structured content flow
- Responsive layout
- Modern typography
- Brand-focused aesthetic

## Built With
- HTML
- CSS

This project focuses on clean design, visual hierarchy, and creating a premium digital presence for a niche café audience.
